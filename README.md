
The chip8-rusterpreter is an emulator for chip8 language written in rust, which comes along with a debugger.

## Run debugger with
````
cargo run -p debugger -- <chip8-file>
````

## Run emulator with
````
cargo run -p chip8
````

## Run tests with
````
cargo test -p <package>
````
